Collection of EL functions for string manipulation: of:abbreviate(), of:concat(), of:prettyURL(), of:encodeURL() and of:escapeJS().

Abbreviate a long string: #{of:abbreviate(stringsBean.string1, 20)}

Concat 2 strings: #{of:concat(stringsBean.string2, stringsBean.string3)}

Following functions deal with the following string: "#{stringsBean.string4}"

  • Create a pretty URL (part): #{of:prettyURL(stringsBean.string4)} check link URL (don't click :) )
  • URL-encode with UTF-8: #{of:encodeURL(stringsBean.string4)} click link #{param.test}
  • Escape for usage in JavaScript: #{of:escapeJS(stringsBean.string4)} click to see alert
