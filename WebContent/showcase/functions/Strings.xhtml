Collection of EL functions for string manipulation: of:abbreviate(), of:concat(), of:prettyURL(), of:encodeURL() and of:escapeJS().

Abbreviate a long string: #{of:abbreviate(string1, 20)}

Concat 2 strings: #{of:concat(string2, string3)}

Following functions deal with the following string: "#{string4}"

  • Create a pretty URL (part): #{of:prettyURL(string4)} check link URL (don't click :) )
  • URL-encode with UTF-8: #{of:encodeURL(string4)} click link #{param.test}
  • Escape for usage in JavaScript: #{of:escapeJS(string4)} click to see alert
