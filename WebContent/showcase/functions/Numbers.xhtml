Collection of EL functions for number formatting: of:formatBytes(), of:formatCurrency(), of:formatNumber(), and of:formatNumberDefault().

Format a given amount of bytes with IEC binary unit: #{of:formatBytes(1023)}, #{of:formatBytes(1024)}, #{of:formatBytes(500000)}, #{of:formatBytes(1048576)}

Format a number as currency using given currency symbol: #{of:formatCurrency(123.4567, '$')}

Format a number using given pattern: #{of:formatNumber(123.4567, '#.00')}

Format a number using locale-default pattern: #{of:formatNumberDefault(123.4567)}
