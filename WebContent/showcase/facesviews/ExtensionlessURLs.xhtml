FacesViews is a mechanism to use SEO-friendly extensionless URLs in a JSF application without the need to enlist individual Facelet source files in some configuration file.

Instead, Facelets source files can be put into the special /WEB-INF/faces-views directory, from where they will be automatically scanned. See the package documentation for additional details.

Navigate to FacesViews pages without an extension: #{request.contextPath}/viewsdemo

Navigate to FacesViews pages with extension: #{request.contextPath}/viewsdemo.xhtml
