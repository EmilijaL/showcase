Welcome to the OmniFaces Showcase application!

Live examples of the OmniFaces components and utilities can be tried out here.

OmniFaces is a fully open source component library for JSF 2, and additionally this showcase application itself is also fully open source.

The showcase application project homepage is available at code.google.com/p/omnifaces-showcase.
Full source code for all demos is available at code.google.com/p/omnifaces-showcase/source/browse.
