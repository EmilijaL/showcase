Welcome to the OmniFaces Showcase application!

Live examples of the OmniFaces components and utilities can be tried out here. Check out the left side menu for all of 'em!

Please note that OmniFaces is a JSF utility library, not a JSF UI/component library. The UI which you see on this showcase is done by PrimeFaces 5.1, with a little bit of custom CSS.

This showcase shows the SNAPSHOT version of OmniFaces! For the showcase of the stable version of OmniFaces, please go to showcase.omnifaces.org.
